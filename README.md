# StoodyPlannner

## 주제 "StoodyPlanner"

: Study Planner로 사용하기 위한 Todo-list

## 팀원소개

: Pm - 주한슬
  T1 - 김규리
  T2 - 이희진

## MVP
  
  1. Todo 입력을 받아서, 리스트로 보여준다.
  2. 완료, 비완료를 리스트로 구분할 수 있다. 
  3. 리스트를 수정, 삭제할 수 있다. 
  4. 정렬 할 수 있다. (남은기간, 중요도, 최신순)

## 사용한 기술
  
  - HTML
  - CSS
  - Javascript
